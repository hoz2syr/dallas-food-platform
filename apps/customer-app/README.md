# Customer App (واجهة العملاء)

واجهة تفاعلية حديثة لعملاء المطعم (ويب/موبايل)، تشمل:
- الصفحة الرئيسية
- القائمة التفاعلية
- تفاصيل المنتج
- سلة التسوق
- الدفع والتأكيد
- تتبع الطلب
- الحساب الشخصي
- الطلبات السابقة
- العروض والتقييمات

## الهيكل الأولي
- `src/pages/` — صفحات رئيسية
- `src/components/` — مكونات مشتركة (بطاقات، أزرار، إلخ)
- `src/styles/` — أنماط CSS/SCSS
- `src/api/` — استدعاءات API
- `public/` — صور وأيقونات

## للبدء
- سيتم بناء كل صفحة ومكون بشكل منفصل مع دعم الترجمة والتصميم المتجاوب.
